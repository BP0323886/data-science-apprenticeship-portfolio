# Submission Notes

The Impact Evaluation component has not been uploaded to GitHub because it contains confidential organisational material and is therefore submitted separately via Turnitin only.

This approach ensures:
- compliance with confidentiality requirements
- separation of public portfolio work from restricted organisational work
- continued reuse of this repository as an evolving apprenticeship portfolio
